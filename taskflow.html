First-time setup — connect your database
Before you start 1. Go to supabase.com → create free account → New Project
2. In SQL Editor, run this once:

create table tasks (
  id text primary key,
  data jsonb not null,
  updated_at timestamptz default now()
);

insert into tasks (id, data)
values ('main', '{"tasks":[],"templates":[],"lastRollover":null}');
3. Go to Settings → API → copy your URL and anon key below.
